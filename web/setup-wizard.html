Welcome to AgentShroud

🔒

AgentShroud is an AI agent security framework that provides comprehensive monitoring, audit trails, and protective measures for your AI systems.

✅ What You Get

  • Real-time security monitoring
  • 26 security modules
  • Comprehensive audit trails
  • Agent trust management
  • Kill switch protection
  • Resource usage tracking

📋 Prerequisites

  • Docker or Podman installed
  • 4GB+ available RAM
  • 2GB+ disk space
  • Network access for container pulls
  • Optional: 1Password service account
Installation Time: Approximately 5-10 minutes depending on your connection speed.

🔐 1Password Service Account

Recommended: Secure, centralized secret management

Requires 1Password Business account

📁 Manual Secret Files

Store secrets in local files

For testing or air-gapped environments
Setting up 1Password Service Account:
  1. Go to your 1Password Business console
  2. Navigate to Integrations → Service Accounts
  3. Create a new service account named "AgentShroud"
  4. Grant access to the vault containing your credentials
  5. Copy the service account token below
This token will be stored securely and used to access your credentials vault.
Manual Secret Files: Less secure but useful for testing.

Create the following directory structure and files:

mkdir -p ~/.agentshroud/secrets

~/.agentshroud/secrets/openai_api_key - OpenAI API key

~/.agentshroud/secrets/anthropic_api_key - Anthropic API key

~/.agentshroud/secrets/gmail_password - Gmail app password

echo "your-api-key" > ~/.agentshroud/secrets/openai_api_key chmod 600 ~/.agentshroud/secrets/openai_api_key
