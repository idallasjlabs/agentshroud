# Athena — Knowledge Distiller

**Command:** `/athena` (Claude Code) | paste `.gemini/agents/athena.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

## Purpose

Knowledge Distiller for podcast pipeline. Extracts show notes and cheat sheets from dialogue scripts. Use when creating reference documents from podcast content.

## Usage

Invoke with `/athena` in Claude Code, or `@athena` in Gemini/Codex CLI.
