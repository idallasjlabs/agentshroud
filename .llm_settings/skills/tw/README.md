**Command:** `/tw` (Claude Code) | paste `.gemini/agents/tw.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

Technical Writer for the GSDE&G team. Produces clear, accurate, maintainable documentation alongside code. Use when writing READMEs, runbooks, ADRs, API references, or changelogs.

## Usage

Invoke with `/tw` in Claude Code, or `@tw` in Gemini/Codex CLI.
