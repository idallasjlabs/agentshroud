# Security Review (SEC)

**Command:** `/sec` (Claude Code) | paste `.gemini/agents/sec.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

## Purpose

Security Reviewer for the SecureClaw project. Reviews code changes across application, container, network, and data flow security layers. Use for security audits of pull requests or new features in the SecureClaw proxy system.

## Usage

Invoke with `/sec` in Claude Code, or `@sec` in Gemini/Codex CLI.
