# MCP Doctor (MCPM-DOCTOR)

**Command:** `/mcpm-doctor` (Claude Code) | paste `.gemini/agents/mcpm-doctor.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

## Purpose

MCP Diagnostics specialist. Diagnoses and troubleshoots MCP server connectivity and configuration issues for GitHub, Atlassian, and AWS MCP servers. Use when MCP servers fail to connect, authentication errors occur, or configuration issues are suspected.

## Usage

Invoke with `/mcpm-doctor` in Claude Code, or `@mcpm-doctor` in Gemini/Codex CLI.
