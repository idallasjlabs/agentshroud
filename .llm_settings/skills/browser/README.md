# Browser — Secure Browser Automation

**Command:** `/browser` (Claude Code) | paste `.gemini/agents/browser.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

## Purpose

Playwright-powered browser automation with enterprise security controls. Use for web tasks requiring browser interaction (form filling, screenshots, page navigation, data extraction) with built-in URL allowlisting, approval queue integration, audit logging, and sandboxed contexts. Supports JavaScript-heavy sites and dynamic web applications.

## Usage

Invoke with `/browser` in Claude Code, or `@browser` in Gemini/Codex CLI.
