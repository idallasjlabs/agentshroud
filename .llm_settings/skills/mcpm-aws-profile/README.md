# MCP AWS Profile Configuration (MCPM-AWS-PROFILE)

**Command:** `/mcpm-aws-profile` (Claude Code) | paste `.gemini/agents/mcpm-aws-profile.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

AWS Profile Configuration specialist for MCP. Helps configure and switch AWS profiles for MCP operations, including SSO, named profiles, and multi-account access.

## Usage

Invoke with `/mcpm-aws-profile` in Claude Code, or `@mcpm-aws-profile` in Gemini/Codex CLI.
