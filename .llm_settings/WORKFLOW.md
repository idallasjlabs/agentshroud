
# AI Engineering Workflow

This repository implements a full lifecycle AI‑augmented development system.

Lifecycle:

1. Product Discovery
2. Architecture Design
3. Development
4. Testing
5. Security
6. CI/CD
7. Deployment
8. Operations
9. Continuous Improvement

## Core Flow

Idea → product‑agent
Design → architecture‑agent
Development → tdd‑engineer
Testing → qa‑agent
Security → security‑reviewer
Build → ci‑agent
Deploy → release‑agent
Operate → sre‑agent
Improve → kaizen‑agent

Agents can chain together through `.claude/ORCHESTRATOR.md`.
