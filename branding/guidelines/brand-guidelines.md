# AgentShroud Brand Guidelines

## Brand Identity

### Mission
AgentShroud provides enterprise-grade security for AI agents, enabling safe autonomous operation through PII redaction, approval workflows, and comprehensive audit trails.

### Vision
To become the trusted security layer for AI agent deployments, protecting sensitive data while enabling innovation.

### Values
- **Security First** - Never compromise on data protection
- **Transparency** - Open-source approach, clear audit trails
- **Privacy** - PII redaction at the core
- **Trust** - Enterprise-grade reliability
- **Innovation** - Advanced AI safety features

## Brand Personality

### Voice & Tone

**Professional but Approachable**
- Technical accuracy without jargon
- Clear, concise communication
- Confident but not arrogant

**Security-Focused**
- Emphasize protection and safety
- Highlight compliance and standards
- Reference security certifications

**Developer-Friendly**
- Speak the language of developers
- Provide clear technical documentation
- Share code examples and best practices

### Writing Style

**Do:**
- Use active voice
- Be specific and concrete
- Provide examples
- Explain complex concepts simply

**Don't:**
- Use marketing fluff
- Make unsubstantiated claims
- Over-promise capabilities
- Use fear-based messaging

## Logo Usage

### Clear Space
Maintain minimum clear space of 50% of logo height on all sides.

### Minimum Size
- **Digital:** 120px width minimum
- **Print:** 1 inch width minimum

### Backgrounds

**Light Backgrounds**
- Use primary logo with transparent background
- Ensure sufficient contrast (4.5:1 minimum)

**Dark Backgrounds**
- Use white or light variant
- Test visibility across different displays

**Colored Backgrounds**
- Use transparent logo
- Ensure contrast meets accessibility standards

### Prohibited Uses
- ❌ Stretching or distorting
- ❌ Rotating at non-standard angles
- ❌ Applying gradients or effects
- ❌ Changing colors
- ❌ Placing on busy backgrounds
- ❌ Using low-resolution files

## Color Usage

### Primary Color Application
Use AgentShroud Blue (`#8bf0fb`) for:
- Primary CTAs
- Interactive elements
- Logo and brand accents
- Navigation highlights
- Active states

### Accessibility
All color combinations must meet WCAG 2.1 AA standards:
- **Normal text:** 4.5:1 contrast ratio minimum
- **Large text:** 3:1 contrast ratio minimum
- **Interactive elements:** 3:1 against adjacent colors

## Typography in Practice

### Hierarchy
Maintain clear visual hierarchy:
1. Headlines grab attention
2. Subheads organize content
3. Body text provides detail
4. Labels clarify function

### Readability
- **Line length:** 50-75 characters optimal
- **Paragraph spacing:** 1.5x line height
- **Alignment:** Left-aligned for body text
- **Orphans/Widows:** Avoid when possible

## Brand Applications

### Website
- Consistent header/footer
- Logo in top-left
- AgentShroud Blue for CTAs
- Clear navigation hierarchy

### Documentation
- Consistent formatting
- Code syntax highlighting
- Clear section headers
- Practical examples

### Social Media
- Profile images use logo
- Cover images follow brand colors
- Consistent post templates
- Professional imagery

### Presentations
- Use official templates
- Consistent slide layouts
- Brand colors throughout
- Logo on all slides

### Print Materials
- High-resolution logos
- CMYK color profiles
- Professional printing standards
- Consistent margins and gutters

## File Formats & Deliverables

### Logo Files
- **PNG:** Web, digital, presentations (72-300 DPI)
- **SVG:** Scalable, web, print-ready
- **PDF:** Print, professional applications
- **EPS:** Legacy print workflows

### Color Profiles
- **RGB:** Digital/screen applications
- **CMYK:** Print applications
- **Pantone:** Spot color printing (when applicable)

## Brand Approval Process

### Internal Use
- Follow these guidelines
- Use official assets only
- Maintain brand consistency

### External/Partnership Use
- Contact brand team for approval
- Provide context and usage plans
- Use co-branding guidelines when applicable
- Respect both brands equally

### Prohibited Uses
- Political endorsements
- Controversial associations
- Competitive comparisons
- Misleading representations
