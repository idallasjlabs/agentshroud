# AgentShroud Brand Guidelines

This directory contains all official branding assets for AgentShroud.

## Directory Structure

```
branding/
├── logos/              # Logo files in various formats
│   ├── png/           # PNG format logos
│   ├── svg/           # SVG format logos (vector)
│   └── variants/      # Logo variations (light/dark backgrounds, etc.)
├── colors/            # Brand color palettes and swatches
├── typography/        # Font files and typography guidelines

```

## Usage Guidelines

### Do's
- Use official logo files only
- Maintain clear space around the logo
- Ensure sufficient contrast with background
- Use transparent version for maximum flexibility

### Don'ts
- Don't distort or stretch the logo
- Don't change logo colors
- Don't add effects or shadows
- Don't use low-resolution versions

## File Formats

- **PNG**: Use for web, presentations, and digital media
- **SVG**: Use for print, scaling, and web when supported
- **Mockup**: Use for client presentations and showcases
