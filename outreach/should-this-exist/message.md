I'm Isaiah Jefferson — systems architect and long-time Should This Exist listener. Your framework for asking whether a technology's design encodes the values we actually want is something I've tried to apply to my own work.

I'm building **AgentShroud** — an open-source security proxy for autonomous AI agents. The question your show asks is exactly the one I keep returning to while building this: what governance structures do we need before we give AI agents real operational authority over systems?

AgentShroud is my attempt to build that infrastructure: a transparent proxy that sits between AI agents and the systems they interact with, enforcing policy, creating audit trails, and keeping humans in the loop for high-risk actions. Not to stop AI from being useful — but to make sure we can answer the question "what did it do, and why?" when we need to.
