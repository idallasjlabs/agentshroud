I'm building **AgentShroud** — an open-source, security-first transparent proxy for autonomous AI agents (Claude, Gemini, ChatGPT, Codex). As these agents gain real operational authority — reading files, making API calls, executing system commands — the attack surface they introduce is largely unaddressed by existing tooling.

AgentShroud intercepts every tool invocation and enforces policy before it reaches the target system. Core capabilities include prompt injection defense, PII redaction, DNS egress filtering, credential exfiltration blocking, and human-in-the-loop approval for high-risk actions. IEC 62443-aligned, 33 active modules, ≥94% test coverage.

This is the kind of infrastructure story that hasn't been widely told yet — and I think your audience is ready for it.
