I'm building **AgentShroud** — an open-source, security-first transparent proxy for autonomous AI agents. It sits between agents like Claude, Gemini, and ChatGPT and the systems they interact with, enforcing policy on every API call, tool invocation, and egress request before anything reaches the target system.

The governance angle is what makes it different from existing tooling. It's not just blocking bad actions — it's creating an auditable, policy-enforced trust boundary so organizations can actually answer the question: "What did the agent do, and why?"
