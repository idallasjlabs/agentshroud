I'm building **AgentShroud** — an open-source, security-first transparent proxy for autonomous AI agents. The threat surface is significant and largely unaddressed: AI agents like Claude Code, Gemini CLI, and ChatGPT now have real operational authority over file systems, APIs, and infrastructure. AgentShroud provides that boundary: 33 active security modules including prompt injection defense, PII redaction, DNS egress filtering, credential exfiltration blocking, and human-in-the-loop approval queues. IEC 62443-aligned. ≥94% test coverage. No stubs.
