## SECTION 9: IMPORTANT NOTES

### Why Standard Character Mark First

A standard character mark protects the word "AGENTSHROUD" in any presentation — any font, any color, any size, any stylization. This is broader protection than a design mark (logo). You can always file a separate design mark application later for the specific logo.

### TEAS Plus vs. TEAS Standard

| Feature | TEAS Plus | TEAS Standard |
|---------|-----------|---------------|
| Fee per class | $350 | $450 |
| Goods/services | Must use ID Manual descriptions | Custom descriptions allowed |
| Email communication | Required | Required |
| Recommended for | Most applicants | Complex or unusual goods/services |

**Recommendation:** Use TEAS Plus. The ID Manual descriptions above should cover AgentShroud's scope. If the examining attorney needs modification, they will issue an Office Action.

### Trademark vs. Patent

These are separate protections:
- **Trademark** protects the brand name "AGENTSHROUD" — prevents others from using confusingly similar names for similar goods/services
- **Patent** protects the invention (the governance proxy system) — prevents others from making/using/selling the invention
- **Copyright** automatically protects the source code — no registration required (though registration enables statutory damages)

Filing both the trademark and provisional patent simultaneously is the strongest initial IP protection strategy.

### The TM Symbol

You can use the TM symbol (AgentShroud™) immediately without registration. This indicates you claim trademark rights. After federal registration, you can use the (R) symbol (AgentShroud(R)).
