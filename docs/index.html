Get started

Up in four commands

Self-hosted Docker stack. No external cloud dependencies.

bash — macOS (Colima) · Linux (Docker CE)
git clone https://github.com/idallasjlabs/agentshroud.git
cd agentshroud

# Store credentials in your OS backend (prompts interactively)
# Auto-detects: 1Password CLI → macOS Keychain → Linux secret-tool → ~/.agentshroud/secrets/
./docker/setup-secrets.sh store

# Bring up the stack — secrets are extracted ephemerally and cleaned up on exit
scripts/asb up

Prerequisites

  • ✓ Git
  • ✓ Docker Engine (Colima on macOS)
  • ✓ Claude OAuth token sk-ant-oat01-...
  • ✓ Telegram bot token
  • ✓ macOS Keychain (built-in, no setup needed)
  • ✓ Python 3 (for key generation)
  • Optional: 1Password CLI (for teams / shared credentials)

asb commands

asb upStart the stack
asb downStop the stack
asb rebuildRebuild images and restart
asb statusShow container health
asb logs [service]Tail logs
asb testRun test suite inside gateway
