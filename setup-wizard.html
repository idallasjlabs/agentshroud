Step 2: Create Gmail Account

OneClaw needs its own Gmail account (keeps it separate from your personal email).

Create a Gmail account:

  1. Open this link: Gmail Signup →
  2. Choose a username like: yourname-oneclaw@gmail.com
  3. For phone verification, you have options:
    • Use Google Voice (free): Get Number →
    • Or use your regular phone number
  4. Complete the signup and verify your email
  5. Come back here when done!
This is just for your reference - we don't store it

Step 3: Get AI API Key

Choose your AI provider and get an API key.

Get OpenAI API Key:

  1. Open: OpenAI Signup →
  2. Sign up with your OneClaw Gmail account
  3. Go to: API Keys →
  4. Click "Create new secret key"
  5. Name it "OneClaw" and copy the key (starts with sk-)
  6. Add billing: Add $10-20 →

Get Anthropic API Key:

  1. Open: Anthropic Console →
  2. Sign up with your OneClaw Gmail account
  3. Go to "API Keys" → "Create Key"
  4. Copy the key (starts with sk-ant-)
  5. Add credits: Billing → "Add to balance" → $10-20
Your key is safe - it stays on your Mac

Step 4: Optional Features

Set up optional features (you can skip and add these later).

Create Telegram Bot:

  1. Open Telegram app
  2. Search for @BotFather
  3. Send: /newbot
  4. Follow prompts (name: "YourName's Assistant", username: ends with _bot)
  5. Copy the bot token
