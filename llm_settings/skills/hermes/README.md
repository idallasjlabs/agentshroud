# Hermes — Podcast Production Orchestrator

**Command:** `/hermes` (Claude Code) | paste `.gemini/agents/hermes.md` (Gemini/Codex)
**Platforms:** Claude Code (native `/skill`) | Gemini CLI, Codex CLI (paste agent file)

## Purpose

Podcast Production Orchestrator. Coordinates the full podcast production pipeline across all specialized agents. Use to run a complete episode or series from topic to audio output.

## Usage

Invoke with `/hermes` in Claude Code, or `@hermes` in Gemini/Codex CLI.
